How to contribute

We appreciate any contribution you can have on this project: testing & reporting bugs, coding, etc.

For testing & reporting bugs

If you are already using Intellj IDEA as your main development IDE then you just need to install the Google Go Language plugin using the Plugin manager and then trying your hand at writing a go program. If you find anything amiss (annoyances, bugs, misfeatures, etc) you should report them in the go-ide issue tracker. If not then you should follow the go-ide guickstart guide to install it on your machine and report anything you find amiss to out issue tracker.

For coders

You will need to install an IntelliJ IDEA plugin development environment and you will need to be somewhat familiar with Java coding. Fork & checkout the code from go-ide source code, do your business and once you have completed your changes (and run the test cases) send a pull request. We will take a look and if everyhing goes well we will integrate it an do a release. We will also help you with anything we can if you ask for help :)
