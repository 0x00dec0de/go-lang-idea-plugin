<b>func Fprintln(w io.Writer, a ...interface{}) (n <a href="psi_element://builtin#int">int</a>, err <a href="psi_element://builtin#error">error</a>)</b>
<p>Fprintln formats using the default formats for its operands and writes to w.
Spaces are always added between operands and a newline is appended.
It returns the number of bytes written and any write error encountered.
</p>
